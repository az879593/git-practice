## DNS A record

## DNS NS record
找到哪個 DNS server 有該網域的實際 DNS record

## Domain Name vs FQDN vs URL 
Domain Name    
用人類可讀的網域名稱轉為 ip 地址
ex : google.com

FQDN : Fully Qualified Domain Name，完整的網域名字    
ex : www.google.com    

URL   
多了 protocol, parameter, port 等等東西

## 為何要加憑證
